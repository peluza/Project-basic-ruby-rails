# Ruby rails basic project

- Install required packages, in the folder shell

```
./install_packets.sh
```

- Create a new proyect

```
rails new [name the proyect]
```

- Create CRUD

```
rails generate scaffold <name table> <name column>:string  <name column>:integer
```

- Migrate databases

```
rails db:migrate
```

- Run server

```
rails s
```
